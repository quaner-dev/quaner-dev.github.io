# Centos 校时

## 部署 ntpdate 服务

```shell
yum install ntpdate -y
```

| 序号 | ntp 名称                | ntp 地址             |
| ---- | ----------------------- | -------------------- |
| 1    | 国家授时中心 NTP 服务器 | ntp.ntsc.ac.cn       |

| 3    | 国际 NTP 快速授时服务   | cn.pool.ntp.org      |

```shell
ntpdate time.pool.aliyun.com
```
