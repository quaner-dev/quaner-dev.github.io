## 查询裸盘中 SN 号

### lsblk 查询 SN 号

```shell
lsblk -o name,size,model,vendor,serial # 可以直接返回裸盘SN号
```

### lshw 查询 SN 号

```shell
lshw
```

## 查询 Raid 中 SN 号

### 使用 storcli64 工具查询 Raid 情况

```shell
/opt/MegaRAID/storcli/storcli64 /c0 show all
```

### 查询磁盘信息

```shell
/opt/MegaRAID/storcli/storcli64 /c0 /eall /sall show all
```
