## Date

### 显示时间

直接显示当前时间

```shell
date
```

### 格式化显示时间

```shell
date +'%Y-%m-%d %H:%M:%S' # 2025-10-16 11:31:33
```

### 设定时间

```shell
date 20251016
```

### 显示时间戳

#### 显示当前时间戳

输出秒级时间戳，如需使用毫秒级，则乘1000

```shell
date +%s
```

#### 转换特定时间时间戳

```shell
date -d "2025-10-16 11:31:33" +%s
```
