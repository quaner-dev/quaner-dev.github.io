## 停止 k8s 服务

```bash
systemctl stop kubelet
systemctl stop docker
```

## 同步文件夹到执行磁盘下

```bash
rsync -avP fse-sata /arachnid
```

## 启动 k8s

```bash
systemctl start docker
systemctl start kubelet
```
