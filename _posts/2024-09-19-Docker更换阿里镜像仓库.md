## 修改 daemon.json 文件

增加 registry-mirrors 镜像仓库

```json
{
  "registry-mirrors": ["https://rsk59qvc.mirror.aliyuncs.com"]
}
```

## 重启 docker

```shell
systemctl daemon-reload
systemctl restart docker
```
