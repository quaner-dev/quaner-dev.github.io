## 书写顺序

```txt
SELECT -> DISTINCT -> TOP -> FROM -> JOIN -> ON -> WHERE -> GROUP BY -> HAVING -> ORDER BY
```

## 执行顺序

```txt
FROM -> ON -> JOIN -> WHERE -> GROUP BY -> SUM、COUNT、MAX、AVG -> HAVING -> SELECT -> DISTINCT -> ORDER BY -> TOP
```
