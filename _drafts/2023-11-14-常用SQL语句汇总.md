# 20231114_常用SQL语句汇总

## 查询动态向量库

```sql
-- 请在调度业务应用MySQL的数据库face_db中执行以下语句
SELECT a.vector_database_id,
    a.vector_database_name,
    b.algo_factory,
    b.algo_version,
    b.feature_type
FROM face_person_library a
    INNER JOIN face_relation_library_algo b ON a.id = b.lib_id
WHERE a.type = 1
    AND a.is_delete = 0
    AND b.feature_type IS NOT NULL
```

## 统计档案抓拍数量

1. 修改ClusteryyyyMMdd为需要查询的表

```sql
SELECT
    cluster_id,
    count( cluster_id )
FROM
    ClusteryyyyMMdd
GROUP BY
    cluster_id
ORDER BY
    count( cluster_id ) DESC
```

## 查询档案关键帧

1. 修改cluster_id为需查询的档案ID
2. 修改ClusteryyyyMMdd为需查询的表名

```sql
SELECT
    a.uuid,
    a.is_keyrecord,
    b.face_uri,
    b.body_uri,
FROM
    ClusteryyyyMMdd a -- 修改表名
    INNER JOIN Record b ON a.uuid = b.uuid
WHERE
    a.is_keyrecord = 1
    AND a.cluster_id = "xxx" -- 修改cluster_id
```
