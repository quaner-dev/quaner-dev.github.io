# 查询ES数据
URL：http://10.2.2.209:31672/person_dynamic/_search

```json
{
  "query": {
    "bool": {
      "must": [
        {
          "range": {
            "captureTime": {
              "gt": "2023-08-16 13:00:00",
              "lt": "2023-08-16 13:10:00"
            }
          }
        }
      ]
    }
  }
}
```
